Important FAQs

What is the DOM (Document Object Model)?

The DOM is a programming interface for web documents that allows scripts to dynamically access and manipulate the structure, style, and content of HTML or XML documents as objects.

How do you select an HTML element using the DOM?

You can select an element using methods like document.getElementById(), document.getElementsByClassName(), document.getElementsByTagName(), document.querySelector(), and document.querySelectorAll()

What’s the difference between innerHTML and textContent?

innerHTML as grabbing everything inside an element, including any HTML tags.On the other hand, textContent only grabs or changes the text inside an element, completely ignoring any HTML tags.

How can you add an event listener to a DOM element?

You can add an event listener using addEventListener(). For example, if you have a button and want something to happen when it’s clicked, you could write: document.getElementById('button').addEventListener('click',function() { alert('Button clicked!'); });
